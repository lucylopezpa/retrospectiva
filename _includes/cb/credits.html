Créditos Técnicos - CollectionBuilder

Esta colección digital está construida con CollectionBuilder, una plantilla de código abierto para crear sitios web de colecciones y exhibiciones digitales, la cual fue desarrollada por bibliotecarios de la facultad de la Biblioteca de la Universidad de Idaho siguiendo la metodología Lib-Static.

El sitio se creó a partir de la plantilla CollectionBuilder-GH que utiliza el generador de sitios web estáticos Jekyll y GitHub Pages para crear y albergar colecciones y exhibiciones digitales.
